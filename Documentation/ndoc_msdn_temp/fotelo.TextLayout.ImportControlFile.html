TextLayout.ImportControlFile Method 

Deserializes XML control file into TextLayout object.

public void ImportControlFile(
   ref String& strFileName
);

Parameters

strFileName
The file path of the control file.

Remarks

This method deserializes the filename specified into the class structure. The control file is XML formatted. Once the file has been deserialized, a few condition checks are made to ensure that the class will function properly. If these conditions are met, the values are passed into the class. Otherwise, and exception is thrown.
