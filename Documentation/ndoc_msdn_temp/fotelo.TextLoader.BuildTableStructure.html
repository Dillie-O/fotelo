TextLoader.BuildTableStructure Method 

This method builds an empty data table using the column definitions specifed.

private DataTable BuildTableStructure(
   ArrayList alColumnStructure
);

Parameters

alColumnStructure
The collection of column information in the data table.

Return Value

A strongly-typed, empty DataTable based on the column information specified.

Remarks

The ArrayList contains a collection of TextLoader.Column structures. If any errors are encountered during the loading process, a custom exception is thrown.
